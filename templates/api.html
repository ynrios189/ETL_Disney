Disney API Documentation:

                This project's API returns the following information on Disney movies: 

                Movie_Title: object, the title of the Disney movie in the dataset
                Release_Date: object, the movie publication date
                Director: object, name of the director of the movie
                Genre: object, genre of the movie
                MPAA_Rating: object, rating of each movie
                Total_Gross: object, total gross revenue of the movie
                Inflation_Adjusted_Gross: object, adjusted total gross revenue to reflect inflation
